# Vehicle Rental Management System

**Live URL:** https://44-b6-a2.vercel.app  
**Repository:** https://github.com/44fahadhasan/B6A2

## Overview

Vehicle Rental Management System is a backend application built with Node.js, TypeScript, Express.js, and PostgreSQL.  
It provides secure REST APIs for user authentication, vehicle management, booking operations, and rental automation using cron jobs.

## Features

- User Registration & Login (JWT Authentication)
- Role-based Access Control: Admin & Customer
- Vehicle Management
- Booking Management
- Auto-return Cron Job
- Centralized Error Handling

## Technology Stack

### Backend

- Node.js
- TypeScript
- Express.js
- PostgreSQL
- bcryptjs
- jsonwebtoken
- node-cron

## Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/44fahadhasan/B6A2.git
cd B6A2
npm install
npm run dev
```
